Push Service

Create a push message from the server with below steps
  • Ask the BI team to provide a STMO link like this and get the result URL with API Key
  • Fill the info below
  • Wait for 1~15 minutes to see the result(depending on your user size. Could be 1~50K)
stmoUrl Provides the client id from STMO
title Display title for the client
body Display body for the client
displayTimestamp Default is 120 second from now. Display time for the client. Use this tool to get the value. Note: Choose "Local time"
mozMessageId Used to identity the campaign for the BI team
mozMsgBatch Free text. used to identity the user clicks in backend.
imageUrl Display image for the client
pushOpenUrl Same as the one we used in Firebase console.
pushDeepLink Same as the one we used in Firebase console.
